## Penjelasan Fisis Singkat
Hukum Ohm dituliskan secara matematis sebagai:

$V = I R$

Hambatan total pada rangkaian seri:

$R_{\text{total}} = \sum_{i=1}^{n} R_i$

Hambatan total pada rangkaian paralel:

$\frac{1}{R_{\text{total}}} = \sum_{i=1}^{n} \frac{1}{R_i}$
